# Board Builder

Board Builder is a Python application that automatically creates Agile project boards from reusable templates.

## Why I Built This

I'm learning Python, Git, GitHub, and REST APIs by building real software instead of tutorial projects.

This project started by automating Trello board creation and will eventually support reusable templates for software projects, church ministries, and personal planning.

## Current Features

- Create Trello boards
- Create Trello lists
- Create Trello cards (coming next)

## Tech Stack

- Python
- Trello REST API
- Git
- GitHub
